## Welcome to Anonymous

Anonymous is a platform where College students can come and post anonymously . Only Verified students can post anything. Anonymous is completely responsive accross all devices.

## Technology Stack
- ## Frontend
  - ReactJs
  - Redux
  - TailwindCSS
- ## Backend
  - Node.js
  - Express
  - Mongoose
- ## Database
  - MongoDB

- ## Firebase
  - For storing images
